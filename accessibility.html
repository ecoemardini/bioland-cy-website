Accessibility Statement

The BIOLAND-Cy project is committed to making its website accessible in line with the EU Web Accessibility Directive and WCAG 2.1 Level AA.

Measures taken

The site uses semantic HTML, sufficient colour contrast, descriptive link text, full keyboard navigation, alt text on images, and a responsive layout tested across mobile, tablet and desktop. Content is written in clear language with a logical heading structure.

Feedback

If you encounter any accessibility barrier on this site, please contact the Project Coordinator via the contact page so we can address it. We aim to respond within five working days.

Conformance status

This site aims to conform to WCAG 2.1 AA. Conformance is reviewed periodically using automated tools (WAVE, Lighthouse) and manual checks.
